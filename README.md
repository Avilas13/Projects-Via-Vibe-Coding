A collection of projects built via Vibe Coding.

## Projects

### 🔢 Advanced Scientific Calculator

Two fully-featured scientific calculators built with a dark, modern UI:

- **Web-based** – open `scientific-calculator/web/index.html` in any modern browser (no install needed)
- **GUI-based** – run `python scientific-calculator/gui/calculator.py` (Python 3 + tkinter)

**Highlights:** trigonometry (sin/cos/tan + inverses), logarithms (log/ln/log₂), eˣ, powers (x²/x³/xʸ), roots (√/∛), constants (π/e), factorial, modulo, percentage, sign toggle, 5-register memory, DEG/RAD/GRAD angle modes, calculation history, full keyboard support.

👉 See [`scientific-calculator/README.md`](scientific-calculator/README.md) for full documentation.
